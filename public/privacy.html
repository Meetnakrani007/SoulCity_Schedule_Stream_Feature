1. Introduction

Welcome to SoulCity By EchoRP ("we," "our," or "us"). This Privacy Policy explains how we collect, use, and protect information when you use our website and services. We are committed to transparency regarding our data practices.

This application uses YouTube API Services to display publicly available livestream information. By using this application, you are also agreeing to be bound by the Google Privacy Policy.

2. YouTube API Services

This application uses YouTube API Services to fetch and display publicly available YouTube data. Specifically, we access the following types of public data through the YouTube Data API v3:

  • Video Titles — The title of upcoming livestreams
  • Video Thumbnails — Thumbnail images associated with videos
  • Scheduled Start Times — When a livestream is scheduled to begin
  • Channel Names — The name of the YouTube channel hosting the stream
  • Video IDs — Used to link directly to YouTube watch pages
  • Published Dates — When the video listing was created

All data displayed on this site is publicly available on YouTube and is fetched in real-time through official YouTube API endpoints. We do not access any private, unlisted, or user-specific YouTube data.

For more information about YouTube's terms and Google's privacy practices, please visit:

  • YouTube Terms of Service
  • Google Privacy Policy

3. Information We Access, Collect, Store, and Use

We want to be completely transparent about all information, including API Data relating to users, that this API Client accesses, collects, stores, and otherwise uses:

3.1 YouTube API Data (API Data Relating to Users)

Through the YouTube Data API v3, this API Client accesses the following publicly available API Data:

  • Video Titles — The title text of upcoming livestream videos
  • Video Thumbnails — Thumbnail image URLs associated with videos
  • Scheduled Start Times — The date and time when a livestream is scheduled to begin
  • Channel Names — The name of the YouTube channel hosting each stream
  • Video IDs — Unique identifiers used to link directly to YouTube watch pages
  • Published Dates — The date when the video listing was originally created on YouTube
  • Live Broadcast Content Status — Whether a video is upcoming, live, or completed

This API Data is fetched in real-time on each page load. We do not store, cache, or persist any YouTube API Data in our own databases or servers. The data is displayed to the user and then discarded when the user leaves the page.

3.2 Personal User Data

We do not collect, store, or process any personal user data whatsoever:

  • We do not require user accounts, login, or authentication
  • We do not collect names, email addresses, or any personal identifiers from visitors
  • We do not access any private, unlisted, or user-specific YouTube data
  • We do not track user behavior or browsing activity

4. How We Use, Process, and Share Information

4.1 Use and Processing

The publicly available YouTube API Data we access is used and processed solely to:

  • Display a gallery of upcoming Soul City GTA RP livestreams on this website
  • Show stream titles, thumbnails, channel names, and scheduled times as provided by the YouTube API
  • Provide direct links so users can watch streams on YouTube.com

All YouTube API Data is displayed as-is from the YouTube API without modification, derivation, or independent calculation. We do not create derived metrics, computed statistics, or alter YouTube data in any way.

4.2 Sharing

We do not share, sell, rent, trade, or transfer any user information or YouTube API Data to any third parties, whether internal departments or external parties. The YouTube API Data accessed by this application is used exclusively for the purposes described above (displaying publicly available livestream information on this website) and for no other purpose.

5. Cookies, Local Storage, and Device Information

This application does not directly store, access, or collect (nor allow third parties to do so through us) any information directly or indirectly on or from users' devices using cookies, pixels, local storage, session storage, or any similar technology.

Specifically:

  • We do not use cookies for tracking, analytics, advertising, or any other purpose
  • We do not use browser local storage or session storage to persist any data
  • We do not use pixel tags, web beacons, or similar technologies
  • We do not fingerprint devices or browsers

However, third-party services loaded on this site (such as Google Fonts and Font Awesome via Cloudflare CDN) may independently use their own cookies or tracking mechanisms on users' devices as governed by their respective privacy policies. We do not control these third-party technologies:

  • Google Privacy Policy (for Google Fonts)
  • Cloudflare Privacy Policy (for cdnjs / Font Awesome CDN)

6. No Derived Data or Independent Metrics

We do not use YouTube's API to offer independently calculated or derived metrics or data that replace or provide new data that isn't available via YouTube's own services. All information shown on this site comes directly from the YouTube Data API v3 and is presented without modification, computation, or derivation.
